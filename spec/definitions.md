# Glossary and Definitions

## Operational Layers

### Content Layer
The primary semantic carriers (text, media, structured data).

### Presentation Layer
Governs visual form and rendering (CSS, layout, typography).

### Structure Layer
The graph structure of the DOM and hierarchical semantics.

### Interaction Layer
Runtime behavior and event handling.

### Data Layer
Machine-readable information (JSON-LD, Microdata).

### Semantic / Intent Layer
Orthogonal annotations regarding communicative role and domain context.

### Provenance Layer
Traceability, authorship, and authority metadata.
