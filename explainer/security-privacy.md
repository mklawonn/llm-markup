# Security and Privacy Questionnaire

This document addresses the [W3C Security and Privacy Questionnaire](https://www.w3.org/TR/security-privacy-questionnaire/).

## 1. What information does this feature expose?
It exposes intent, policy, and provenance metadata about the content.

## 2. Does this feature expose any personal data?
Only if the author or provenance fields contain personal identifiers (e.g., author names).

## 3. Does this feature introduce new state?
No, it is generally declarative.

## 4. How does this feature interact with the Same-Origin Policy?
The global policy (Header JSON) should be governed by the origin.
